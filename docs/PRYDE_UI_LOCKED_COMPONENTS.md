# Pryde UI Locked Components

The following UI components are considered production-stable
and must not be altered during marketing phase except for
functional bug fixes.

## Locked
- Post cards
- Reaction system
- Comment threads
- Navbar
- Buttons & pills
- Modals
- Quiet Mode

## Guarded
- Messages page (layout only)
- Profile header

## Unlocked
- Settings page
- Auth/legal pages
- Edge-case responsive fixes
